# Утилита *GenDiff* для поиска отличий в конфигурационных файлах
### Возможности утилиты:

* Поддержка разных форматов файлов (*.json*, *.yaml*, *.ini*)
* Генерация отчета в виде *plain*, *stylish* и *json*

```
$ make install
```

```
$ npm link
```
